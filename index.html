개인정보처리방침

Design Your Life (DYL)

시행일: 2026년 3월 30일 | 최종 수정: 2026년 3월 30일

Design Your Life는 사용자의 개인정보를 소중히 여기며, 「개인정보 보호법」에 따라 아래와 같이 개인정보처리방침을 수립하여 공개합니다.

1. 수집하는 개인정보 항목

본 앱은 로컬 퍼스트(Local-First) 아키텍처를 채택하여, 모든 데이터를 사용자 기기 내부에 암호화하여 저장합니다. 서버로 전송되는 개인정보는 없습니다.

선택적 수집 항목 (사용자 동의 시):

  • Google 계정 정보 (이메일, 이름, 프로필 사진) — Google Drive 백업 기능 사용 시
  • Google Calendar 일정 데이터 — 캘린더 동기화 기능 사용 시 (읽기 전용)
  • GitHub Personal Access Token — GitHub 백업 기능 사용 시

자동 수집 항목:

  • 앱 사용 통계 (Firebase Analytics) — 익명화된 사용 패턴
  • 앱 오류 정보 (Firebase Crashlytics) — 크래시 로그 (개인 식별 불가)
  • 광고 식별자 (Google AdMob) — 맞춤 광고 제공

2. 개인정보의 수집 및 이용 목적

  • Google Drive 백업: 사용자 데이터의 클라우드 백업 및 복원
  • Google Calendar: 사용자의 일정을 앱 내 캘린더에 표시 (읽기 전용)
  • GitHub 백업: 사용자 데이터의 GitHub 저장소 백업
  • Firebase Analytics: 앱 품질 개선 및 사용자 경험 향상
  • Firebase Crashlytics: 앱 안정성 모니터링 및 오류 수정
  • Google AdMob: 앱 운영을 위한 광고 표시

3. 개인정보의 보유 및 이용 기간

  • 기기 내 데이터: 사용자가 앱을 삭제하거나 데이터를 초기화할 때까지
  • Google Drive 백업: 사용자가 직접 삭제할 때까지 (Google Drive 내 appdata 폴더)
  • GitHub 백업: 사용자가 직접 삭제할 때까지 (비공개 저장소)
  • Firebase 데이터: Google의 데이터 보존 정책에 따름

4. 개인정보의 제3자 제공

본 앱은 사용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 아래의 경우 예외로 합니다:

  • 사용자가 Google Drive/Calendar 연동에 명시적으로 동의한 경우 (Google LLC)
  • 사용자가 GitHub 백업 기능을 명시적으로 활성화한 경우 (GitHub Inc.)
  • 법률에 의해 요구되는 경우

5. 개인정보의 파기

  • 앱 삭제 시 기기 내 모든 데이터가 자동으로 파기됩니다.
  • 설정 > 계정관리 > 데이터 초기화를 통해 수동으로 파기할 수 있습니다.
  • Google Drive 및 GitHub 백업 데이터는 사용자가 직접 삭제해야 합니다.

6. 데이터 암호화

기기 내 모든 사용자 데이터는 AES-256 암호화로 보호됩니다. 암호화 키는 iOS Keychain / Android Keystore에 안전하게 저장됩니다.

7. 광고

본 앱은 Google AdMob을 통해 광고를 표시합니다. 광고 표시 시 Google의 광고 정책이 적용됩니다.

  • 인터스티셜 광고 (전면 광고): 메모 저장 시 표시
  • 보상형 광고: 클라우드 백업 및 데일리 리추얼 완료 시 표시
  • macOS 및 Windows에서는 광고가 표시되지 않습니다.

8. 이용자의 권리

  • Google 계정 연결 해제: 설정 > 계정관리에서 언제든지 연결을 해제할 수 있습니다.
  • 데이터 삭제: 앱 삭제 또는 설정 > 데이터 초기화를 통해 가능합니다.
  • 광고 개인화 설정: 기기의 Google 설정에서 광고 개인화를 끌 수 있습니다.

9. 아동의 개인정보 보호

본 앱은 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 만 14세 미만 사용자는 법정대리인의 동의 하에 서비스를 이용해야 합니다.

10. 개인정보처리방침의 변경

본 방침이 변경되는 경우, 앱 내 공지 또는 본 페이지를 통해 변경 사항을 안내합니다.
